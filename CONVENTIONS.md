# CONVENTIONS

## 規範/Guideline

本專案中所有表示「任意型別」的情境一律使用 **any**，不使用 **interface{}**。  
In this project, whenever we need to represent “any type,” we always use **any** instead of **interface{}**.

### handlers pakage

除非有特殊情況，否則我們堅持一個指令一個文件的原則。  
We adhere to the principle of one command one file, unless there are special circumstances.
